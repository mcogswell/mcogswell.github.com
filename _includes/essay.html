Introduction

Markdown is a simple way to format text so it can accurately and precisely convey meaning while maintaining properties desired by advanced computer users. It came about as a way to make life easier for people who not only spend their days working at computers, but also work in fields about computers.

When working with computers, as in most fields, one needs to do a lot of writing, but in this field one needs to do a lot of writing in special environments. Specifically, the writing needs to happen on a computer (typed in with the keyboard) and it needs to be easily dealt with by many different computers.

Operating Description

Markdown is written in plain text, which means the text does not need to be processed to take its final form. Writing markdown is just typing text into an editor while following certain rules. Markdown is just formatted text.

Description of Parts and their Function

What follows are some examples of text written with Markdown formatting and how that same text appears when translated to HTML and rendered in a browser.

Emphasis

Most authors are used to using bold or itallics to add meaning to documents written on their computer. Markdown syntax allows you to do this in plain text by surrounding things you want to make bold in pairs of underscores and by surrounding thing you want to make itallic in single underscores.

Headings

In Markdown, headings are simply created by adding hash marks before the section name; level one and two headers can be created by adding lines of equal signs or dashes, respectively. This provides a way to create longer, yet still well organized plain text documents.

Lists

Both ordered and unordered lists can be created by preceeding each item with a number, asterisk, or dash, just like many people do in emails.
